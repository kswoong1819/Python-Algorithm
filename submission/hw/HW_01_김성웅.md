   따라서, 아래의 값을 비교하기 위해 작성해야하는 코드를 작성하세요.

   ​    a = 0.1 * 3

   ​	b = 0.3

   ```py
   a=0.1*3
   b=0.3
   
   print(a == b)	# False
   
   print((a-b) <= 1e-10)	# True
   
   import sys
   print((a-b) <= sys.float_info.epsilon)	#True
   
   import math
   print(math.isclose(a,b))	#True
   ```
