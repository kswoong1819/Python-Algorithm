3. 다음과 같은 이차방적식이 있을 때 근을 찾는 수식을 파이썬 코드를 이용하여 출력해 보시오.

   ```python
   import math
   
   def second_order_direct(a,b,c):
       if b**2 - 4*a*c < 0:
           print ('No answer')
       else:
           x1 = (-b + math.sqrt(b**2 - 4*a*c)) / (2*a)
           x2 = (-b - math.sqrt(b**2 - 4*a*c)) / (2*a)
           print(x1, x2)
       return x1, x2
   
   second_order_direct(1,4,-21)
   ```
