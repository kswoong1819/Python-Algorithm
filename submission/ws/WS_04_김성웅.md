1. 양의 정수 x를 입력 받아 제곱근의 근사값을 반환하는 함수를 작성하세요.

   ```python
   def my_sqrt(number):
   
    left_end , right_end = 1,number
       result = 1
       while abs(result**2 - number) > 0.00001:
           result = (left_end + right_end) /2
           if result**2 < number:
               left_end = result
           else:
               right_end = result
       return result
   
   num = int(input())
   print(my_sqrt(num))
   ```
